紅魔館の住民の誰かとして遊べるモードを搭載した紅NTR16です
本当はパッチとして製作したつもりなのですが、
住民版あなたを動かすに当たっていろいろいじっていたら昔のデータに不具合ががががが

そんな事でこのパッチを導入した場合、前までのデータは使用不可能となります。
使用する場合はそこを考慮することをお奨めします。

なお、同梱してある本体である別人版も前までのデータを使用することはできません。ご了承ください。
