era紅魔館protoNTR用
	「着替えコマンド修正パッチ」

パッチの簡単な説明
	あなた私室で行えるコマンド「[430]着替える」が上手く動作しない不具合を修正します

導入手順
	「era紅魔館protoNTR0025」に「era紅魔館protoNTR0025用パッチまとめ」の「16_05_28」をあてた後
	ERBフォルダを入れてください

アップロード日
	2016/06/17

修正・改造など
	修正・バランス調整はご自由に


〜パッチ内容〜
	私室でのみ行える「[430]着替える」において、同室しているキャラがいると「あなた」を選択できず着替えられない不具合と
	本来、同室しているキャラがいない場合は即座に「あなた」を着替えさせる処理が呼び出されるはずが
	一度同室しているキャラがいる状態で「[430]着替える」を行うと、その後同室しているキャラが居ない場合でも
	居る時と同じ処理がされていた不具合を修正します。
	
	また、数か所の条件式を簡略化しました。(たぶん不具合は起きないと思いますが何か問題があったら修正をお願いします)
