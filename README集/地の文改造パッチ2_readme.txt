アイテム[腟鏡]
	6000￥ 一点モノ
	
	女性の膣腔内に挿入し開大して膣腔を検診したりするための医療用具
	よくクスコって呼ばれてるあれ
	大きいのから小さいのまでサイズが一式揃っている
	
	うふふ時にコマンド[膣鏡]を使用できるようになる
	
コマンド[腟鏡]
	使用条件
		・アイテム膣鏡購入済み
		・対象に女性器があり、処女ではない
		・対象が恋慕を取得済み
		・縄、バイブを使用中ではない
		・下半身上着、下着を脱いでいる
			但しスカートなどの下着を脱がせるタイプの上着は除く
		・場所別条件
			各寝室（条件無し）
			厨房、各トイレ（好感度1500以上）
			食堂、図書室、大浴場（好感度2600以上）
			野外、館内通り道(好感度3700以上)
			
			妖精達の家での条件は各寝室と同じ
		・潤滑Lv3以上、それ以下の場合ローションを持っていること
			但し厨房ではローションは不要

	使用条件
		・腟鏡使用中
