ただし渡すためには相手が親愛持ちまたは恋慕持ちで一定以上が必要となります。
（結婚指輪を通販で買うとかどうなのよと思うけどそこはご都合主義ってものです）

男で人妻は要するに主夫ってやつですね。一応男にも指輪は渡せます。
（現実世界でも同性婚の話はあるけど指輪とか渡すのかな）

あと申し訳ないですが、私が作った段階では本当にただ人妻属性を付けるだけのパッチです。
