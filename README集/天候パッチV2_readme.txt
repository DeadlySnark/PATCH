体力気力減少処理
	減少条件
		野外に居る（正門、庭、湖）
		気温がキャラの耐久温度を超える
	減少量
		基本減少量 = (耐久温度-現在気温の差)
		※現在気温とはコマンド実行前の気温とする
		それに (経過時間 - 経過時間/5の余り) / 5 の値を掛ける
		
		気力が残っている場合
			体力の減少値 = 減少量/2
			気力の減少値 = 減少量
		気力が０の場合
			体力の減少値 = 減少量

	最大減少量（チルノ耐暑換算）
	　最も熱い時に最も時間が経過する
		最大最高気温   39(基礎最高気温31＋最大変動値3＋異常気象:温度+5)
		チルノ暑さ耐性 20
		基礎減少量     39 - 20 = 19
		コマンドによる最大経過時間 庭:戦闘訓練60
		減少値 19 * (60 - 60%5<0>) / 5 = 228
		
	　とはいっても実際に体力気力が228減少するにはこの気温で戦闘訓練を行い、
	　且つ60分後にチルノが野外に居ることが前提
	
	
	簡単に言えば夏の暑い時期や冬の寒い時期に外へ出たり外で行動起こしたり
	野外プレイしたりするといつもより体力気力の減少が激しいよってこと
	あと5分未満の行動を起こした場合この処理は行われない
