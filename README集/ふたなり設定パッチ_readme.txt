Q.なにこれ？
A.女性キャラクターに生やしたり消したりします。既存のアイテム購入では一人ずつ指定しますが、このパッチで追加されたメニューでは「初期メンバー全員を50%の確率でふたなりにする」とか「娘全員を100%の確率でふたなりにする」のような指定になります。

Q.適用したほうがいいの？
A.必要な物ではありません。「日によって生やしたり消したりしたい。全員バラバラのタイミングで。」とか「娘が増えてきたけど、今から全員ふたなりに変更したいなあ」とか「ふたなり母親が実の娘に毎日無理やり逆レイプされるばかりではなく、たまには娘の精液を中出しされる側になってもいいのではないか」という人だけ使えばいいです。逆に、以下に挙げる人は適用しないほうが良いでしょう。１．ふたなり住民に興味がない人 ２．ふたなりにするかどうか一人ずつ指定するのが正当であり唯一の方法であるべきだという主義の人 ３．後述の「Q.副作用とか欠点とかは？」のAが気になる人 ４．手厚いサポートがあって当然だと思ってる人

Q.パッチ適用方法は？
A.0.027のSHOP.ERBに上書きして下さい。

Q.どこで使う機能？
A.起床やセーブロードなどを選択するメニューの一番下に [500] - 一括ふたなり が追加されてます。そこからGO。

Q.副作用とか欠点とかは？
A.[500] - 一括ふたなり を使わない場合：コードが少し増える分、Emueraの負荷が少し増えるだろうけどそれだけだと思います。[500] - 一括ふたなり を使う場合：一人だけ変化させるアイテムは有料なのに一括変更は無料というのは変かもしれません。その他、不具合も存在するかもしれません。
