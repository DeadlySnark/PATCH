簡単なパッチの説明
	訪問者の移動嗜好に、「誰かいる場所」をより好むよう仕掛けをしました。
	寝取りラストスパート機能もついています。

導入手順
	"era紅魔館protoNTR"に"era紅魔館protoNTR不具合修正パッチ130630"を適用した後に
	このNTR.ERBを上書きしてください
	
	
主な追加処理
	●屈服度に応じて、移動場所に補正をかける。
	　→ただし、訪問者が先に、住人が後に動くので、空移動になる場合もある。
	　→MOVEMENT.ERB側で対応してください＞誰か
	●スレ159 >>150氏の短気化処理を追加。
	●無情なるデバッグ用分岐。初日から訪問者が出現します。
	　→正規のNTRに反映する際には消した方が良いです。
