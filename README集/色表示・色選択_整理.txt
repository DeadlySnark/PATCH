[era紅魔館protoNTR0015用　色表示/選択整理パッチ]

髪・肌・目の色表示および選択時でのこと。
キャラクターと訪問者のステータスを参照する時、色が文字だけではわかりづらいかと思ったので、
色パターンの■を表示するようにしてみました。

あと、色の選択時、文字数の整理がされておらず選び辛かったので、
見やすくなるように修正してみました。


肌の色について
小麦色と褐色の番号が被っていたので
濃褐色を削除、小麦を2、褐色を3に割り当てました
肌の白については、真っ白ではなく青みがかった白がいいと思い、アリスブルーにしました。
