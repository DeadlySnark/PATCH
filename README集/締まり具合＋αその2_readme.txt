				フラグ値  射精影響  快Ｖ/Ａ影響
	ぎちぎち     0~99      X*N         100%
	きゅっきゅっ 100~249   X*(N-1)      85%
	きゅうきゅう 250~449   X*(N-3)      70%
	がばがば     450~699   X*(N-5)      55%
	ぽっかり     700~      X*(N-7)      40%

	
	初期は全員ぎちぎちからスタート
	
	フラグの数値が大きくなるほど緩くなり
		・挿入する側の射精ゲージが上がりにくくなる
		・挿入された側の快ソースが上がりにくくなる
