■お出かけ判定詳細
	
	・[妊娠中]、[育児中]はお出かけしない
	・[浮気公認]4LV以上、または[公衆便所]がなければお出かけしない
	・貞操帯の鍵を持っている場合はお出かけしない
	
	①放置時間を「現在時刻-最後に訪問者とセックスした日」で算出
	②基本の休暇時間を13日(18720)とする
	③[公衆便所]かつ[親愛]なら休暇時間を半分に
	④[公衆便所]なら休暇時間に子宮内精液量(%)を乗算
		※子宮内に精液が残ってない場合休暇時間は0になるため、必ず客を取りに行く
	⑤[恋慕]がある場合は休暇時間10%増し
	⑥[淫乱]、[浮気癖]、[浮気な蜜壺]、[浮気な尻穴]、[浮気な唇]により休暇時間10%ずつ減少（累乗）
		※[娼婦]の場合は10%ではなく20%休暇時間減少
	⑦浮気快楽刻印の平方根*10%休暇時間減少(最大34%減)
	⑧CFLAG:性欲 だけ休暇時間減少
	⑨放置時間が休暇時間より大きくなればお出かけする

メモ
	子宮内の精液は3日でなくなる、つまり[公衆便所]は最低３日に１回はお仕事しなければならない
	[恋慕][公衆便所]は毎日鍵を買って目一杯中出ししてればそうそうお出かけしない(自分の精液も計算される)
		…娼婦紋付いてる？それはもう諦めてお仕事に送り出しなさい
	素質と刻印をコンプリートしていた場合は休暇時間が34%(4日と少し)まで減る。
		基本的には以前(3～14日、浮気快楽刻印1LV毎に1日減)より出かけづらくなっているはず
