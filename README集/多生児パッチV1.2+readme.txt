-era紅魔館protoNTR用多生児出産と(ryパッチ V1.2+-


簡単なパッチの説明
	親や子供、兄弟の関係を能力表示画面に表示したり
	ついでに合性も表示したり
	双子とか三つ子とかが産まれるようになるパッチ
	

導入手順
	"era紅魔館protoNTR Ver.0.005"に"era紅魔館protoNTR修正パッチ130828"を適用した後に
	このパッチを入れてください
	
V1.2+追加・変更ファイル
	DIM.ERH
	
	PREGNACY_S.ERB 
	多生児パッチ.ERB
	
主な追加処理
▲v1.2+
	●親別出産数の最大配列数の増加(300:300 → 305:305)
	●親別出産数増加時の客、訪問者用の分岐を追加
	●出産表の客を不明に変更
	

▲v1.2
	●DIM.ERHファイルの追加
	●里子も含む出産した子供の数を父親別に保存・表示
	●ステータス画面で一度に表示する量が多くなる為ページ送り機能を追加
	　1ページ目に既存の能力情報を、2,3ページ目に本パッチの情報を表示
	　一部情報は特定のキャラであったり出産を経験しないと表示されません
	●SHOPからステータス画面を表示した際にステータスを表示するキャラを変更するコマンドが表示される

▲v1.1
	●キャラが妊娠している場合、能力表示で誰の子を孕んでいるかが確認できるように
	●出産時の永琳診療所から私室へ戻す処理を修正
	●VerUP処理に前Verでの不具合を修正
		・出産後の診療所から戻る処理ができていないキャラを修正する
		(臨月時以外で診療所に滞在する処理が出来ない限りはこの処理で大丈夫なはず)
		
▲それ以前の旧パッチ
	●多生児出産処理を追加、確率で2人以上の子供を同時に産むように
	　最大で８つ子が生まれます
	　多生児出産時に里子に出す場合、その時産まれた子供を全員里子に出します
	
	●能力表示画面に表示された対象キャラの親子・兄弟関係及び合性の情報を追加
	　・兄弟の情報は生まれが早い順に表示される(名前の前に番号がふられる)
	　　対象は黄色で表示
	　　特殊な兄弟の場合には名前の後ろに()で表示、異母兄弟なら(異母)、異父兄弟なら(異父)
	　　対象と同じ時に生まれた兄弟は()内に"双子"や"三つ子"などの同時に生まれた人数分の多生児表示がなされる
	　　
	　　兄弟に対象以外の多生児が存在する場合、その多生児達に()の表示はされない
	　　里子にでた兄弟がいた場合、その兄弟は表示されない
	　　
	　・親の情報は親の名前と()内に親から見た何人目の子であるかが表示される
	　　
	　・子供の情報は父親としての子供と母親としての子供を表示する
	　　()内は子供の母<父>の名前
	　　ふたなりなどでキャラが父母どちらにもなりえるのでこんな風にしてみた
	　　
	　　その下に里子に出した人数とそれらをひっくるめた対象の子供の合計人数を表示
	　　

このパッチで使用している追加フラグ
	FLAG:53 このパッチのVerUP用フラグ
		追加フラグの調整
		
	FLAG:54 訪問者の子供の数
		訪問者はCFLAG使えないのでこのフラグに保存
		
	CFLAG:85 子供の人数
	CFLAG:86 父親の何人目の子供(子供用フラグ)
	CFLAG:87 母親の何人目の子供(子供用フラグ)
	CFLAG:88 里子に出した数
	
	CFLAG:353 何人出産するか(出産後は育児中の人数となる)
	CFLAG:354 多生児の兄弟用フラグ
	
	DIM.ERH
	　親別出産数:X:Y 母親Xの産んだ父親Yの子供の数
