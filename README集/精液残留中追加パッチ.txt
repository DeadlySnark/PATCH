―――era紅魔館protoNTR用 性交後の精液残留中追加パッチ―――

※説明
	ターゲットが訪問者と性交から一定時間経った時に
	ターゲットと同じ場所にあなたか訪問者が居る場合、秘所から精液をこぼすのを表示します。
	一定時間内に再び訪問者と性交した場合はやり直しになります。

	時間経過で消えるので、分かりやすくローター関連と同じ場所に配置。
	バイブとかの追加があるかも知れないので、連番の4番では無く10番にしています。

※導入方法
	『 era紅魔館protoNTR002』に『era紅魔館protoNTR不具合修正パッチ130711』
　　　　を適用した後にこのパッチを入れてください

※変更・追加・削除一覧
	CFLAG.CSV
		CFLAG:29  精液残留中

	NTR.ERB
		1711行目・2037行目
		性交開始時に精液残留中の終了処理を追加
		CFLAG:ARG:精液残留中 = 0

		2730行目・2753行目

		CALL NTR_PUNISHMENT_10(ARG)

		3135行目
		精液残留中の開始処理を追加
		@NTR_PUNISHMENT_10(ARG)

	NTR.SOURCE
		105行目
		精液残留中の終了処理を追加
		IF CFLAG:LOCAL:精液残留中
