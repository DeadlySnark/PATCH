-era紅魔館protoNTR用地の文とか一部変更・追加パッチ-


簡単なパッチの説明
	・一部コマンドを追加したり地の文の変更をしたりした(詳しくは主な追加処理に記載)
	・相手をしているキャラの服装が表示されるようになる

導入手順
	"era紅魔館protoNTR Ver.0.007"に"era紅魔館protoNTR修正パッチ131018"を適用した後に
	このパッチを入れてください
	
主な追加処理
	コマンドを追加
		[正常位させる]
			場所によって[対面立位させる]になる
		
		[後背位させる]
			場所によって[背面立位させる]になる
	
	
	一部コマンドにおける地の文の変更
		[逆レイプ]
			自分の子供に行った場合地の文が変化する
			
		[上半身上着脱衣]
			上半身下着を着けていない場合に地の文が追加される（自慰中は除く）
			
		[下半身上着脱衣]
			下半身下着を着けていない場合に地の文が追加される（自慰中は除く）
			
		[上半身下着脱衣]
			全体的に地の文を変更
			快Ｂの度合によって地の文が追加される
			胸の大きさによって地の文が追加される（男性は除く）
			
		[下半身下着脱衣]
			女性は濡れ具合によって、男性は勃起具合によって地の文が変わる
		
	射精時に親子関係によって地の文が変化
		
ファイルの変更・追加・削除一覧
	Str.CSV,Train.CSV
		正常位させる、後背位させるを追加
	COMABLE.ERB
		1501行目あたり
			追加コマンドの実行判定追加
	EVENT_MESSAGE_COM.ERB
		1673行目あたり
			追加コマンドの地の文追加
		コマンド[上半身上着脱衣][下半身上着脱衣][上半身下着脱衣][下半身下着脱衣]の部分
			地の文の変更
	EVENT_MESSAGE_ORGASM.ERB
		56,134 188行目辺り
			親子関係により地の文が変化するように
	INFO.ERB
		126行目
			対象の衣服の状態を表示する呼び出しを追加
		一番下
			↑の呼び出しにおける処理を追加
	USERCOM.ERB
		34行目あたりから
			追加コマンドの場所によるコマンド名変更処理を追加
	
ついでに
	客汎用地の文.ERBにて一部同じ条件の分岐が二つあったのを修正
	加えて一部に口上ランダム生成を追加(汎用口上ランダム生成.ERB)
