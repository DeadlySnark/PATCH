era紅魔館protoNTR_1クリックでTARGET選択パッチ

----------------------------------------
使用したバージョン
----------------------------------------
本体　　　：era紅魔館protoNTR0006
修正パッチ：era紅魔館protoNTR修正パッチ130908

----------------------------------------
説明
----------------------------------------
相手を変える[802]を使わずにターゲットの変更を可能にします

1クリック選択には、『ここにいる』以降に表示されるキャラクターの名前をクリックしてください

----------------------------------------
改変したファイルと箇所
----------------------------------------
◆USERCOM.ERB
	@USERCOM
	159行目から3行追加
	入力　−　700　の値をTFLAG:104に代入
	上記処理のため700番台のコマンドを占有する
	ついでにタブがずれてたので修正

◆INFO.ERB
	@INFO_SetTrainMode(ARG:0)
	920行目から『;同じ場所にいる他のキャラクターを表示』の部分
	PRINTFORMをPRINTBUTTONに変更
	ボタンの番号は(700 + キャラ登録番号)
	他多少変わっているのは強調表示しようとした名残り
	強調表示は見送ったけど、スッキリしたのでそのままに
