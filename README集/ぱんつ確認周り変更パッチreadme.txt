era紅魔館protoNTR用
	「ぱんつ確認周り変更パッチ」

パッチの簡単な説明
	1.うふふ中にキャラの服を脱がせた際、パンツ確認フラグが立たなかった部分を修正しました
	2.覗きをした際、ワンピースでもパンチラが起こるようになりました
	3.会話拡張「今日のぱんつについて」でCFLAG:ぱんつ確認が1(脱がせて確認した)だった場合にもぱんつについての話が出来るようになりました

	「era紅魔館protoNTR0023」に「era紅魔館protoNTR0023用パッチまとめ」の「15_11_07」をあてた後に
	ERBフォルダを入れてください

アップロード日
	2015/11/11

修正・改造など
	修正・改造・バランス調整はご自由に


〜パッチ内容〜
	1.うふふ時に下半身上着を脱がせた際などに「FLAG:雑多設定,1」(ズボンなどを穿いている時下着の情報が得られなくなる設定)がONになっていないと
	　ぱんつ確認フラグが立たなくなっていた部分を修正しました。
	　これにより設定がONになっていない場合でも脱がすことが出来れば「CFLAG:ぱんつ確認」が1になり、下着がステータスに表示されるようになります。

	　また、上下一体型の上着(ワンピースなど)を脱がせた際にも判定が出なかったので一体型の服に限り上半身上着を脱がせた時に確認判定が出るように修正しました。


	2.コマンド「覗く」をした場合、今までは特定の場所で裳以外のスカートを履いている場合のみパンチラが起こっていましたが
	　ロングスカートがパンチラするならワンピースもパンチラくらいするだろうと思いワンピースの場合にもパンチラが発生するように変更しました。

	　チャイナ服もパンチラさせるか迷いましたが、あれはロングタイトスカートに分類されると思うので流石に下から覗いて見えるものでもないと思い追加しませんでした。


	3.コマンド「会話」で一定条件を満たしたときに発生する選択肢「今日のぱんつについて」は
	　今までは覗いてパンツを確認した場合にしかこの選択肢は発生しませんでしたが
	　上記のうふふ中に脱がせた場合の確認判定でも発生するように変更しました。

	　それに伴い、「CFLAG:ぱんつ確認が1(脱がせて確認)かつ普通の下着」の場合に表示される文を追加しました。
	　ただ、紅魔館の住人達がパンツの話をしただけであなたを咎めるかどうか微妙なので
	　文をしっくりくる感じに変更できる方がいたらぜひ修正をお願いします。
