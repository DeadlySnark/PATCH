-era紅魔館protoNTR用妊娠処理変更パッチ V1.0-

簡単なパッチの説明
	現在
		・排卵日に膣内射精した際、妊娠チェックが入る
	これ
		・膣内射精時に射精したキャラの精子をストックする
		・一日終了時に排卵日で子宮内に誰かの精子が有れば妊娠する確率が出る
		・妊娠確率は子宮内の精子量によって変化する
	
導入手順
	"era紅魔館protoNTR v 003"に"era紅魔館protoNTR不具合修正パッチ130715"を適用した後に
	このパッチを入れてください
	
	
このパッチで使用する追加フラグ

	CFLAG:350 妊娠確率における分母(便宜上ここやERB内では"子宮内体積"と書いてある)

	FLAG:53 このパッチのVerUP用フラグ
		子宮内体積のフラグを既存のキャラに初期追加する際に使用
		
	TFLAG:220 射精される精子量

	DA:X:Y = Xの子宮内にあるYの精子総量
	TA:X:Y:Z = Xの子宮内にあるYのZ番の精子量
	TB:X:Y:Z = Xの子宮内にあるYのZ番の精子の残り生存日数
	
変更点
	射精関連
		・射精時に 精力の値＝精子量 として取得する
		　訪問者などの登録外のキャラは固定値からランダムで精子量を取得
		・膣内射精を行った場合、TA:X:Y:Zに精子量が加わる
		・ストックされた精子は3日間生存し、1日ごとに残量の20％が減少する
		・精子は1日ごとに別のフラグ(TA:X:Y:Z)にストックする
		
	妊娠関係
		・妊娠判定の処理は一日終了時に行う
		　対象が排卵日で誰かの精子が子宮内に残っていれば妊娠する確率の処理が発生する
		　総合精子量が多いほど妊娠する確率が上がる
		・妊娠が確定した際に多人数の精子が子宮内に有れば出産するまで
		　誰の子か分からないようになる
		

	呼び出し PREGNACY_S_CHECK をコメントアウト

妊娠確率
	確率は子宮内の総合精子量/子宮内体積
	体型が普通の場合、排卵日に精力が1000(現在の精力の最大値)の状態で膣内射精を１回行うと２０％の確率で妊娠する
	子宮内体積は体型により変化し、幼児体型など通常より小さければ子宮内体積も小さくなり、妊娠しやすくなる
	　逆に長身など体が大きければ妊娠しにくい

生理機能追加パッチ等debug.ERB
	テスト用ファイル、ERBフォルダに入れれば生理周期や妊娠判定、射精時の精子量、キャラの子宮内の精子量等が分かるようになる
	それらの文字はピンク色で表示されます
	
その他
	VariableSize.CSVにて二次,三次元配列型変数を追加,数値増加変更をしているので若干重くなるかも？
	このパッチで使われている二次,三次元配列型変数の整理処理は作ってません
	　その為DELCHARA等で登録キャラを削除するような事がある場合、不具合を起こす可能性があります
	処理の都合上、290人目以降の子供が誰かに膣内射精を行うとエラーを起こす
