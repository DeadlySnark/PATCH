-era紅魔館protoNTR用 夏のリファクタリング祭り-

簡単なパッチの説明
	このパッチは、最新版emuera1813用です。（1739bでは動作しません）
	CFLAGやらを使ってERBを読みやすくしたつもりのものです。
	ということで、動作は変わらないはずです。きっと。おそらく。

導入手順
	"era紅魔館protoNTR0003"にCSV,ERBフォルダを上書きしてください
	
	
主な追加処理
	●「変数一覧.txt」をベースに「CFLAG.csv」、「FLAG.csv」に変数名を定義。
	●多数のERBファイルで、「FLAG:21」等の数字ベースの記述から、
	　「FLAG:訪問者の現在位置」等の内容ベースの記述に変更。
	●IF〜ELSEIF〜ELSE〜ENDIFの表記をSELECTCASE〜ENDSELECTに変更。（NTR.ERBのみ）
	●口上の呼び出しを、TRYCALLFORMからTRYCALLLISTに変更。（NTR.ERBのみ）
	●パチュリー口上、チルノ口上のバグ取り
	●動作は変わっていないはず。そんな気がします。

このパッチで使用している追加フラグ
	ありません。ただし、表記を大きく変えています。
	（１ヶ月後に、こんな汚いソースを書いたのは誰だ！と激怒するフラグを自分自身に付与しました）
